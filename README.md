## VKSIMPLEPLAYER

Плеер музыки с сайта Вконтакте на WPF.

**Цель:** Знакомство с паттерном MVVM и библиотекой Prism

**Использовано:**

 - Prism
 - MEF
 - VkNet
 - MahApps
 - MaterialDesignThemes
